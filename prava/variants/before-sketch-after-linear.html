The Journey

Stop Managing Permissions · Start Sharing With People

1
The Problem

The Assignment

Email client needs file sharing. The technical specification called for implementing "Read, Write, Delete" permissions with access control matrices.

Original requirements: Technical permission system

But users don't think about "permissions matrices". They think: "I need to share this with Anna and Petr."

2
Exploration

Paper Sketches

Hand-drawn iterations exploring different approaches. The key insight: shift from managing permissions to finding people.

Iteration 1: Card Grid

3x3 grid of people cards with ON/OFF toggles. "Doporučené" (Recommended) section at top. Focus: Visual selection of people first.

Iteration 2: Enhanced Columns

Added structure: Name | ON/OFF | Share columns. One card highlighted to show selection state. Testing different layouts.

Iteration 3: Contact List

Simplified to contact names (JAN, FRANTA, TONDA, KAREL). Experimenting with list vs. card format. Search field prominent.

"Stop managing rights. Start finding people."
3
The Solution

We Removed the Complexity

A working solution that puts people first. Type a name, click, share. That's it.

The final solution: Search → Select → Share

What we removed:

  • Technical jargon (RWD → "Who can edit?")
  • Complex permission matrices
  • The concept of "rights management"
  • Everything between the user and their goal

What we added:

  • Search and autocomplete
  • Visual feedback (pills, highlights)
  • Smart defaults
  • Human language

The Journey

Finding what to remove is more valuable than deciding what to add.

Aleš Brom · UX Designer
